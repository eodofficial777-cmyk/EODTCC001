# **App Name**: EOD 終端機

## Core Features:

- 使用者註冊與驗證: 允許使用者使用獨特的角色名稱、噗浪資訊和其他必要詳細資訊進行註冊，並使用 Firebase Authentication 進行身份驗證。 帳戶需要經過管理員批准後才能啟用登入。
- 儀表板: 顯示使用者的角色資訊、屬性、資源、裝備、近期任務和導航連結。
- 角色名冊: 顯示所有已批准角色的可搜尋和篩選清單，可按陣營、種族和榮譽點排序。
- 任務提交: 允許使用者透過網址提交創意任務（美術作品、寫作等），對任務進行分類並授予適當的榮譽點。
- 陣營對抗資訊: 顯示即時陣營分數，同時考量玩家人數，以提供與情境相關的優勢提示。 只有管理員才能重置賽季積分。
- 陣營專屬商店: 提供陣營專屬的商店介面，用於使用遊戲內貨幣購買物品。 原子性地更新庫存並檢查是否有足夠的貨幣。
- 共鬥戰場: 顯示共鬥戰場、陣營災獸血量攻擊力、玩家血量屬性、可裝備物品、使用道具、可使用技能。每20秒為一回合。
- 管理員後台管理功能: 1.帳號審核：帳號啟用、帳號停用(無法登入)、角色陣營轉換、角色種族轉換等 2.任務管理：新增任務類型、新增任務、審核任務等 3.共鬥管理：開啟共鬥(需命名共鬥戰場名稱，開啟後倒數30分鐘)、新增陣營怪物、共鬥log存檔 4.陣營對抗管理：重置陣營積分開啟新賽季、留存當期積分 5.商店道具管理：新增商品、設定商品種類、商品上架下架管理 6.裝備合成管理：某裝備+某物品=新裝備(合成後原裝備會消失) 7.技能管理：分陣營、分種族各自有不同既能運用在共鬥上 8.稱號管理：有一般稱號跟隱藏稱號，只要達成就會顯示，可以在儀表板上更換 9.特殊獎勵發放：透過複合篩選的條件，如陣營、種族、期間繳交任務次數、參與共鬥次數、持有金錢、榮譽點數等，來發放特殊獎勵道具、稱號、榮譽點或貨幣

## Style Guidelines:

- 主要顏色：柔和的橙色 (#D97925)，在黑色背景中保持熱情的色調。
- 背景顏色：黑色 (#000000)。
- 強調顏色：灰白色 (#CACACA)，以產生醒目的效果。
- 內文字體和標題字體：'Inter'，一種無襯線字體。
- 程式碼字體：'Source Code Pro'，用於顯示程式碼片段。
- 圖示：簡單、基於線條的單色圖示。
- 版面：每個元件的模組化設計。
- 在網站的不同部分變更內容時，使用微妙的動畫。
